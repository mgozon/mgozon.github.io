Life Organizer

Javascript
React
Tailwind CSS
Firebase

A simple tool to organize the projects of your life

Life Organizer is a web app that provides a way to manage all of the projects an individual has going on in their life. It is built using a tree, allowing for users to categorize projects within whatever nested categories makes sense to them. The history page displays all of the projects created as well as the tasks associated with them, and their data is stored using firebase.


The intent of this app was to help me reflect every week on what I had accomplished and so I could improve on my methods the following week. Pencil and paper did not work well since there are too many projects and tasks to keep track without the convenience of a hierarchy like a tree.
